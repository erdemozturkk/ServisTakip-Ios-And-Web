# ServisTakip

Flutter ile hazırlanmış servis takip uygulaması. Bu repository proje dosyalarını içerir.

Kısa kurulum:

1. Flutter yüklü olduğundan emin olun.
2. Bağımlılıkları yükleyin:

```bash
flutter pub get
```

3. Uygulamayı çalıştırın:

```bash
flutter run
```

Notlar:
- Harita ve konum özellikleri için `google_maps_flutter` ve `location` paketlerini kullanmaktadır.
- iOS/Android platform ayarlarını (API key, Info.plist, AndroidManifest) yapmayı unutmayın.

GitHub'a yüklemek (özet):

1. Yerel repo oluşturun ve ilk commit'i yapın:

```bash
cd PATH/TO/flutter_application_1
git init
git add .
git commit -m "Initial commit: ServisTakip"
```

2. GitHub'da `ServisTakip` adında bir repo oluşturun (veya `gh repo create ServisTakip --public --source=. --remote=origin` kullanın).

3. Uzaktaki repoya push edin:

```bash
git branch -M main
git remote add origin https://github.com/<kullanici_adiniz>/ServisTakip.git
git push -u origin main
```

Eğer isterseniz bu adımları sizin için terminalde çalıştırabilirim (GitHub erişim bilgilerin/izinlerin varsa).
